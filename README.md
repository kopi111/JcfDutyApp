# JCF Traffic Offense Manager (Android)

Jamaica Constabulary Force mobile application for managing traffic offenses and violations.

## Features
- **Traffic Offense Database**: Comprehensive violation catalog
- **Offense Descriptions**: Detailed information for each violation
- **Traffic Code Sections**: Legal code references
- **SQLite Integration**: Local database for offline access
- **Search Functionality**: Quick offense lookup
- **Officer Reference Tool**: Field-ready information

## Technology Stack
- Java (Android)
- Android SDK
- SQLite database
- Connection Helper pattern

## Use Cases
- Traffic enforcement
- Citation reference
- Officer training
- Legal code lookup
- Field operations

## Target Users
Jamaica Constabulary Force traffic officers and law enforcement personnel.
